# PS2SMB-Utils
A collection of utils for use with PS2SMB when using Free McBoot

## PopStarter ELFs Batcher

Place this .bat file inside of the `PS2SMB/POPS` folder.

Optionally make a shortcut and put in the top level PS2SMB directory, for convenience.

Whenever adding new .VCD to the directory, just re-run this batch file.
